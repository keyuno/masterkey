Create Event Assets

Go to counterwallet.io and open a wallet. To create assets you will need bitcoin. Choose "Address Action" and select "Create a Token (Asset)". Choose a name for your asset. To create numeric assets you will only need bitcoin, if you want to use letters you will also need XCP. Next you will need to write the event description.

Event Description

In the description field write the description of the event. Choose a common description of the event. For xample: "Symphony No. 41, Mozart". If necessary make an additional transaction to complete the description. Keep the description short and simple. Set "Quantity" to "1", uncheck "make divisible" box and create the asset. Finish the process by selecting "Lock Token Issuance".
